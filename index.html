Last Projects

Some of the last games I worked on.

ChessD-19

ChessD-19 is a puzzle game where you control a chess piece and must help to save your world.

The game was developed for a GameJam in 10 days with the theme "stay at home". the theme was explored in the idea that being in a house on the board your piece remains safe.

The game has 15 levels, storyline, cutscenes, dialogs, customization, upgrades and special moves.

As the game was developed in a short time and for a Brazilian audience, the game is entirely in Portuguese, but it can be played through your browser here.

ChessD-19 was developed with the tools Blender, Gimp and Unity3D in C # language and published for Windows and HTML5.

Ninjas vs Zombies

Ninjas vs Zombies is a turn-based strategy game on a small board, which requires calculating each move very well, so that it is not the last one. You control one of the few ninjas surviving a zombie apocalypse and must collect fruits to help your companions.The mechanics of the game are simple, just drag your finger in the direction you want to walk and each movement alternates between walking attacking and walking preparing the next attack.

The game was developed with the tools Gimp, Krita, Spriter and Unity3D in C#. Implemented Appodeal API and published for Android on Google Play.

QuizGuaruja3D

QuizGuaruja3D is an educational electronic game with manual mechanics. Here students encourage order and education while learning more about the city of Guaruja.

Unlike other electronic board games, in QuizGuaruja3D, the game does not warn you when you walk, roll the dice or draw a card, all actions can be done at any time. Passing naturalness and discipline to the players.

This game was developed for an academic teacher of the educational school system in order to move on to other teachers an activity class work following the guidelines of the city.

The project was developed with the tools Gimp and Unity3D using the C # language and compiled for Windows.
